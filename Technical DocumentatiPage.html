JavaScript and Java

JavaScript and Java are two different programming languages with some similarities.

console.log('Hello World');

JavaScript is mainly used for web development, while Java is used for building larger, more complex applications.

System.out.println("Hello World");

JavaScript is interpreted, while Java is compiled.

  • JavaScript is often used in front-end web development.
  • Java is known for its cross-platform capabilities.
  • JavaScript supports functional programming.
  • Java is statically typed, JavaScript is dynamically typed.
  • JavaScript runs in browsers, while Java requires a virtual machine (JVM).
HTML and CSS

HTML (HyperText Markup Language) is used to structure web pages, while CSS (Cascading Style Sheets) is used for styling them.

<html>

HTML defines elements like paragraphs, headings, links, etc.

<h1>Heading</h1>

CSS is used to style those elements with colors, fonts, and layout.

  • HTML defines the structure of a webpage.
  • CSS defines the appearance of a webpage.
  • HTML tags are the building blocks of webpages.
  • CSS selectors are used to target HTML elements.
  • CSS can be inline, internal, or external.
Python

Python is a high-level, interpreted programming language known for its simplicity and readability.

print("Hello, World!")

Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

  • Python is easy to learn and has a clean syntax.
  • It is widely used in web development, data science, and automation.
  • Python has a vast ecosystem of libraries and frameworks.
  • It is a cross-platform language, which runs on multiple operating systems.
  • Python is dynamically typed, meaning you don't need to declare variable types explicitly.
Data Structures

Data structures are ways to organize and store data for efficient processing.

list = [1, 2, 3, 4]

Common data structures include arrays, linked lists, stacks, queues, and trees.

  • Arrays allow access by index.
  • Linked lists are useful for efficient insertions and deletions.
  • Stacks follow the Last In First Out (LIFO) principle.
  • Queues follow the First In First Out (FIFO) principle.
  • Trees represent hierarchical data.
Algorithms

Algorithms are step-by-step instructions for solving problems or performing tasks.

def bubbleSort(arr):

Some common algorithms include sorting algorithms like quicksort, mergesort, and bubble sort.

  • Quicksort is an efficient sorting algorithm with average time complexity of O(n log n).
  • Mergesort divides the input into smaller subarrays and merges them back together.
  • Bubble sort repeatedly swaps adjacent elements if they are in the wrong order.
  • Algorithms can be classified based on time and space complexity.
  • Greedy algorithms make local optimal choices to find a global optimum.
